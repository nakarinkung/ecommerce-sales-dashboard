# Data Analytics Project: E-commerce Sales Performance Dashboard
## Overview
This project involves the development of a comprehensive E-commerce Sales Dashboard to track and analyze business performance metrics. The dashboard provides real-time insights into sales trends, profit margins, and product performance across different regions and categories, enabling data-driven decision making for business optimization.

## Dataset
Source: E-commerce Sales Data

Description: Contains transactional data including sales, profits, quantities, and customer information

Key Dimensions: Product categories, customer regions, shipping types, time periods

Metrics: Sales revenue, profit margins, quantity sold, year-over-year growth

## Tools & Technologies
Data Visualization: Microsoft Power BI

Data Analysis: DAX (Data Analysis Expressions)

Business Intelligence: KPI tracking, comparative analysis, trend identification

## Project Steps
1. Data Integration & Preparation
Consolidated multiple data sources into a unified data model

Cleaned and transformed raw transactional data for analysis

Established relationships between dimensions and fact tables

2. KPI Development & Calculation
Created key performance indicators (KPIs) for comprehensive business monitoring:

Year-to-Date (YTD) Sales: $11.53M (0.83% growth)

YTD Profit: $1.34M (4.50% growth)

YTD Quantity: 107.2K units (-7.29% decline)

YTD Profit Margin: 11.58% (5.37% improvement)
